## Repro steps (client-friendly)
1. Start dev server
   - `npm run dev -- --hostname 127.0.0.1 --port 3003`
2. One-click demo ingest
   - `POST /api/demo/ingest`
   - Or click **Demo** in the UI
3. Ask a question
   - `POST /api/query` with `{ query }`
